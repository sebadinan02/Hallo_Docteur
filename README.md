# Le site hallodocteur-burkina.com est un site vitrine pour la clinique Hallo_Docteur,
# un portail d'accès à certains services qu'offrent la clinique qui peuvent s'effectuer en ligne.
#

# Pour accéder à ces services, il faut tout simplement se rendre sur le site hallodocteur-burkina.com.
#  On arrive ainsi sur la page d'accueil du site
# La page d'accueil en paticulier met l'accés sur la localisation de la clinique,
#  le service de conseils en ligne ainsi que son service de consulations en ligne,
#  et pour finir les acrualités de la clinique et une section pour quelques conseils 
# et informations relevant du domaine médical.
#
# De plus l'on a accès à un menu qui donne accès à quatre options à savoir : 
#        => La Clinique
#        => Spécialités
#		 => Contactez-Nous
#        => Prendre RDV
#
# ==================== La Clinique =======================
# 
# En cliquant sur cette option, on est redirigé vers une page du même intitulé 
# qui présente la clinique Hallo Docteur, son historique compéte, les différents
# services dont elle dispose, son matériel technique et bien toute l'équipe médicale
# de la clinique qui oeuvre à offrir aux patients les meilleurs soins pour un rétablissement optimal.
# 
#
# ==================== Spécialités =======================
#
# Un clic sur cette option redirige vers la page Spécialités qui liste les différentes spécialités 
# qui meublent le mur des compétences de la clinique. Par spécialités, l'on entend les différentes 
# offres et services sur lesquelles s'imposent la clinique dans le domaine du médicale, ce qui fait
# sa différence des autres cliniques du pays. La clinique dispose ainsi de plusieurs particularités que
# l'on découvre sur cette page sous forme de "carte à spécialités"
#
#
# ==================== Contactez-Nous =======================
#
# Cette page dispose d'un formulaire de saisie qui permet ainsi de contacter 
# directement le service d'accueil de la clinique pour toute personne voulant 
# contacter la clinique pour un besoin quelconque de renseignements ou une suggestion 
#
#
# ==================== Prendre RDV =======================
#
# Sur cette page, l'on trouvera deux formulaires : l'un pour toute personne voulant prendre RDV ave un medecin
# pour un problème de santé et/ou une visite médicale de routine, l'autre concerne
# tout acteur du domaine médical ou professionnel de la santé desirant offrir ses services 
# à la Clinique Hallo Docteur. Par professionnels, l'on entend entre autres les pharmacies, les médecins 
# externes à la clinique (médecin moderne ou traditionnel).
